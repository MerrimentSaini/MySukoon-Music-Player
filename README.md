A modern music player built with React, Vite, Node.js, and the Jamendo API.

## ✨ Features
- 🎵 Stream music
- 🔍 Search songs and artists
- ❤️ Favorites
- 📂 Playlists
- 🌙 Responsive UI
- ⚡ Fast loading

## 🛠 Tech Stack
- React
- Vite
- JavaScript
- Node.js
- Jamendo API

## 🚀 Installation

npm install

npm run dev
